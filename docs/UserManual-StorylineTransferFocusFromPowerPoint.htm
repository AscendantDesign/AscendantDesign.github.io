Storyline - Transfer Focus Order From PowerPoint Slide

Quickly apply PowerPoint focus order on your Storyline slide.

 

Background Information

Storyline ignores the tab order of items in PowerPoint slide whenever you import that slide into a project. However, in cases where you have already intentionally set that tab order within the original PowerPoint file, a single Ambassador-64 action can restore it for you in Storyline.

 

Instructions

Follow these steps to restore PowerPoint focus order on your Storyline slide.

•        Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.

•        In Ambassador, select the PowerPoint Tools panel.

•        Click Open File.

•        Open the PowerPoint file containing the original slide.

•        Either select the original slide directly in the PowerPoint application or click Select Slide to select it in Ambassador.

•        Select Ambassador's Storyline Tools panel.

•        Click Open File.

•        Open the Storyline file containing the slides to be updated.

•        Click OK.

•        Click Select Slide.

•        Select the slide to update and click OK.

•        In the Active Command drop-down list, select Transfer Focus Order From PowerPoint Slide.

•        Click Run Command.

•        The following message will be displayed.
When you click OK, the focus order of objects on the selected PowerPoint slide will be applied to corresponding objects on the selected Storyline slide.

•        Click OK.

•        Click Save File to store the changes to the Storyline file.

•        Click Close File to release the file back to the Storyline application.
