PowerPoint - Position Objects

Amplify your fine-tuning capabilities by re-applying present locations and dimensions over multiple slides and multiple shapes.

 

Typical Applications

It is not uncommon to work on PowerPoint decks where shapes need to be aligned at some symmetrical point that is common among multiple slides or sized equally to fit within a certain predefined area.

For example, maybe the text on a slide should always start at 2.34" to the left of the page, or maybe the per-page illustration should always be 4.32" x 3.21". The Position Objects activity is used to quickly and accurately achieve these goals.

 

Process Example

We were recently designing a website layout in PowerPoint, and as you might imagine, a lot of HTML layout is based upon pixels instead of inches. However, using Ambassador's Position Objects function, we were able to use this simple process to immediately ready the pixel-specific sizes and locations of every object on the page.

  • Select the PowerPoint Tools panel.
  • Click Open File.
  • Select the file to open and click Open.
  • Select the slide containing the objects to inspect.
  • In the Active Command drop-down list, select Position Objects.
  • Click Run Command.
  • In the Units group, make sure the Pixels setting is checked.
  • Take the following steps for every object on the page.

 

Per-object steps.

  • In PowerPoint, select an object.
  • In Ambassador, click Get Coordinates.
  • Read the X, Y, Width, and Height values.

 

 

Activity Description

During the Position Objects activity, you can select different objects, move in different directions, change priority midstream, and activate different types of changes in an ad-hoc fashion. As a result, there is not necessarily a specific linear method for operating this panel, other than to click the Apply button whenever you want to store the selected location and dimensions to the currently selected object, then to click Close when you are finished editing.

The following illustration and descriptions are provided as a general reference.
