Storyline - Set Focus Order

Quickly set the focus order for all shapes on a slide.

 

Background Information

Setting focus order might be one of the least convenient tasks to accomplish in Storyline. The only method Storyline makes available for focus order activity is a drag-and-drop list, and only one item can be dragged at a time while organizing. As the number of items to be sorted grows, the time required to sort those items becomes exponentially slower.

 

Mentions of this problem can be found in the Storyline discussion titled ' Focus Order Question' at: https://community.articulate.com/discussions/articulate-storyline/focus-order-question.

 

Use Ambassador's Set Focus Order action to drastically reduce the amount of time needed to correct the focus order of all of the shapes on a slide.

 

Instructions

Follow these steps to adjust the focus order of shapes on a slide.

  • Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.
  • In Ambassador, select the Storyline Tools panel.
  • Click Open File.
  • Open the Storyline file containing the slides to be updated.
  • Click OK.
  • Click Select Slide.
  • Select the slide to update and click OK.
  • In the Active Command drop-down list, select Set Focus Order.
  • Click Run Command.
  • Use any combination of these tactics to rearrange the focus order of items on the slide.
  • Click a column name on the header row to sort the list.
  • Set the values of each row in the index column.
  • Click the Clear Sort button to remove any sorting attributes from the list.
  • Use the Layer filter list to view items by layer or simultaneously for all layers.
  • [Click] and drag an item to reposition it in the list.
  • [Ctrl][Click] to toggle-select items to drag.
  • [Shift][Click] to extend-select items to drag.
  • Click the Renumber button to finalize the current order.
  • Click OK.
  • Click Save File to commit the changes to the Storyline file.
  • Click Close File to release the file to Storyline.

 

Activity Description

Following is an example of the Set Focus Order panel.

 

Key

Name

Description

A

Header Row

Click any of the column headers to sort by that column.

B

Shapes List

List of shapes. The Index column is editable.

C

Layer Filter

When blank, entries from all layers are visible. When a value is selected, only items from that layer are displayed.

D

Renumber

Renumber all index values in the list from top to bottom in the current order.

E

Clear Sort

Clear the column header sorts, reverting to a current natural view of the list layout.

F

Cancel

Return to the Storyline Tools panel without applying any changes.

G

OK

Apply current changes and return to the Storyline Tools panel.
