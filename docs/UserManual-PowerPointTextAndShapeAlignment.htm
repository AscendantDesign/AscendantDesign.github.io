PowerPoint - Text and Shape Alignment

Alternate method for quickly aligning multiple shapes.

 

Typical Applications

PowerPoint has very specific methods for aligning and distributing objects on a page, often based in the average or extent of all selected objects. For example, when aligning top on the following slide, the alignment procedure will select the topmost object as the reference, then bring all other items to the top, as shown in the following before and after figure.

Instructions For Aligning With Anchor Shape

To align or distribute shapes, follow these steps.

•        In the PowerPoint Tools panel, open a file and select the slide containing the shapes you wish to position.

•        In the Active Command drop-down list, select Text and Shape Alignment.

•        Click Run Command.

•        In the Shapes list, first select the object you will treat as the anchor. Then, use [Ctrl][Click] or [Shift][Click] as appropriate to select the other items to be positioned or distributed.

•        If you are going to be aligning the objects, check the Align checkbox then select the type of alignment to apply.

•        If you are going to be distributing the objects, check the Distribute checkbox and select the Horizontal or Vertical option.

•        Click OK to apply the changes and return to the PowerPoint Tools panel.

Activity Description

The Text and Shape Alignment panel contains multiple controls for aligning and distributing shapes on the current slide.

 

 

Key

Name

Description

A

Shapes Selection

Select one or more shapes to align or distribute on the current slide. If using the Anchor method, the first item selected will become the anchor.

B

Anchor reference point

If selected, the first item selected in the Shapes list will become the anchor upon which shape alignment is measured.

C

Anchor name

Shape name of the first-selected shape in the Shapes list.

D

Left active

When checked, all aligned items will be horizontally fixed at the specified coordinate.

E

Left coordinate

The X coordinate at which to horizontally align items.

F

Top active

When checked, all aligned items will be vertically fixed at the specified coordinate.

G

Top coordinate

The Y coordinate at which to horizontally align items.

H

Align active

When checked, an alignment of the selected shapes will be performed. The choices are:

-       Top. Align selected items to the top location.

-       Middle. Align selected items to the middle of the anchor or coordinate.

-       Bottom. Align selected items to the bottom of the anchor or vertical coordinate.

-       Left. Align selected items to the left edge of the anchor or horizontal coordinate.

-       Center. Align selected items to the center of the anchor object or horizontal coordinate.

-       Right. Align selected items to the right side of the anchor object or horizontal coordinate.

I

Distribute active

When checked, the selected objects will be distributed vertically or horizontally, depending upon the internal selection. Note that when using Align and Distribute in the same action, a horizontal distribution can only be performed on vertical alignments, and visa-versa. This is by design.

J

Cancel

Close the current panel without making any changes and return to the PowerPoint Tools panel.

K

OK

Apply the current changes and return to the PowerPoint Tools panel.
