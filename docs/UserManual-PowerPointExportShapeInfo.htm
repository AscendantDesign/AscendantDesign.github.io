PowerPoint - Export Shape Info To CSV

Interact with and manage shape information in a spreadsheet application.

 

Background Information

Use the shape to CSV export function to save information about the shapes on the currently selected slide.

The following columns are present in the data.

Name

Data Type

Description

Name

Text

Name of the shape.

Left

Number

Left coordinate of the shape, in inches.

Top

Number

Top coordinate of the shape, in inches.

Width

Number

Width of the shape, in inches.

Height

Number

Height of the shape, in inches.

ZOrder

Number

Shapes with higher ZOrder numbers are drawn in front of shapes with lower ZOrder numbers.

 

General Instructions

Follow these steps to use the action.

•        Assure that you have an open PowerPoint file.

•        Select the slide you wish to export.

•        In the Active Command drop-down, select Export Shape Info to CSV.

•        Click Run Command.

•        Set the destination filename and click Save.
