Storyline - Set Timeline Names

Establish clear names for shapes imported into a project to ease the task of bulk management.

 

Background Information

After importing a slide into Storyline from PowerPoint, none of the timeline names for the newly imported objects will be set. One of the clearest examples of this effect can be seen when you open the Home / Focus Order dialog, as suggested in the following illustration.

 

Using the Set Timeline Names action will initialize all uninitialized names in the project to values that not only stand apart from one another, but are also equal to their original names in the PowerPoint slide, making it easy to track assets from PowerPoint through to a Storyline project.

 

Instructions

Follow these steps to initialize timeline names on a slide.

•        Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.

•        In Ambassador, select the Storyline Tools panel.

•        Click Open File.

•        Open the Storyline file containing the slides to be updated.

•        Click OK.

•        Click Select Slide.

•        Select the slide to initialize and click OK.

•        In the Active Command drop-down list, select Set Timeline Names.

•        Click Run Command.

•        The following prompt will be displayed.
When you click OK, all names on the Storyline timeline that have not been set to a user-defined value will be initialized to their current natural names.

This has the effect of making the Focus Order list much easier to read within the Storyline application.

•        Click OK.

•        Click Save File to store the changes to the Storyline file.

•        Click Close File to release the file for use with the Storyline application.
