Using Storyline Tools

Productivity aids for Articulate Storyline.

 

Action Index

The Storyline Tools section of Ambassador-64 is reserved for actions that work directly upon files and projects created in Articulate Storyline. In this toolbox, you will find the following actions.

•        [Enable Video Compression Drop-Down](UserManual-StorylineEnableVideoCompression).

•        [Export Focus Order To CSV File](UserManual-StorylineExportFocusOrderToCSV).

•        [Export Media References To CSV File](UserManual-StorylineExportMediaReferencesToCSV).

•        [Import Focus Order From CSV File](UserManual-StorylineImportFocusOrderFromCSV).

•        [Import Media References From CSV File](UserManual-StorylineImportMediaReferencesFromCSV).

•        [Improve Video](UserManual-StorylineImproveVideo).

•        [Refresh Media From Sources](UserManual-StorylineRefreshMedia).

•        [Set Alternate Text](UserManual-StorylineSetAlternateText).

•        [Set Focus Order](UserManual-StorylineSetFocusOrder).

•        [Set Timeline Names](UserManual-StorylineSetTimelineNames).

•        [Transfer Focus Order From PowerPoint Slide](UserManual-StorylineTransferFocusFromPowerPoint).

 

General Instructions

Most actions work about the same way, as described in the following steps.

•        Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.

•        In Ambassador, select the Storyline Tools panel.

•        Click Open File.

•        Open the Storyline file you will be working on.

•        Click OK.

•        In many cases, the action will only apply to a single slide. In those cases, click Select Slide, select the slide to operate upon, and click OK.

•        In the Active Command drop-down list, select the name of the action you intend to run.

•        Click Run Command.

•        Perform any activities, selections, or configuration related to the action.

•        Click OK.

•        Click Save File to commit the changes to the Storyline file.

•        Click Close File to release the file to Storyline.
