Storyline - Set Alternate Text

Quickly set alternate text on one or more shapes.

 

Background Information

Although Storyline allows adjustment of alternate text for shapes, the user interface for that setting is inconvenient and slow. Especially when hundreds of alternate text descriptions will need to be set in a single session, the task might take multiple hours to complete in Storyline.

Using the Set Alternate Text action, all of the alternate text values for the shapes in a slide can be set within a few seconds to a few minutes.

 

Instructions

Follow these steps to set the alternate text descriptions for all shapes on a slide.

•        Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.

•        In Ambassador, select the Storyline Tools panel.

•        Click Open File.

•        Open the Storyline file containing the slides to be updated.

•        Click OK.

•        Click Select Slide.

•        Select the slide to update and click OK.

•        In the Active Command drop-down list, select Set Alternate Text.

•        Click Run Command.

•        Enter the alternate text descriptions for each shape in the Alt Text column.

•        Click OK.

•        Click Save File to commit the changes to the Storyline file.

•        Click Close File to release the file to Storyline.

 

 

Activity Description

Following is a visual example of the Set Alternate Text panel, along with descriptions of the various controls.

 

Key

Name

Description

A

Header Row

Click any of the column headers to sort by that column.

B

Shapes List

List of shapes. The Alt Text column is editable.

C

Layer Filter

When blank, entries from all layers are visible. When a value is selected, only items from that layer are displayed.

D

Cancel

Cancel the operation and return to the Storyline Tools panel without making any changes.

E

OK

Apply all of the changes made to alternate text entries and return to the Storyline Tools panel.
