PowerPoint Tools

 

 

Introduction

All of the commands on the PowerPoint Tools section require, at least, an open PowerPoint file. Follow these steps to open your file.

  • In the navigation bar at the left, select the PowerPoint Tools section.
  • Click Open File.
  • Navigate to the folder where your file is located.
  • Select the file and click Open.

The currently selected Slide will be displayed in the Active Slide textbox, but you can also select another slide directly in PowerPoint, or by clicking the Select Slide button.

 

Action Index

Once a file is open and a slide is selected, the following commands are available in the command list. Select one of these and click Run Command.

  • Convert Placeholders to Textboxes.
  • Export Shape Info to CSV.
  • Import Shape Info from CSV.
  • Position Objects.
  • Quick Animation.
  • Remove Bullet From Paragraphs.
  • Text and Shape Alignment.

 

General Instructions

Most actions work about the same way, as described in the following steps.

  • Save and close other instances of your PowerPoint file.
  • In Ambassador, select the PowerPoint Tools panel.
  • Click Open File.
  • Open the PowerPoint file you will be working on.
  • Click OK.
  • In many cases, the action will only apply to a single slide. In those cases, click Select Slide, select the slide to operate upon, and click OK. Alternatively, the current PowerPoint slide can be selected directly in PowerPoint.
  • In the Active Command drop-down list, select the name of the action you intend to run.
  • Click Run Command.
  • Perform any activities, selections, or configuration related to the action.
  • Click OK.
  • Click Save File to commit the changes to the PowerPoint file.
  • Click Close File to close PowerPoint.
