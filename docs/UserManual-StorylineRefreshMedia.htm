Storyline - Refresh Media From Sources

Refresh one or more media files in your Storyline project directly from updated source media without having to open Storyline.

 

Background Information

Storyline only allows one media file to be reloaded from its original source at a time. Even with working with the Media Library, each file has to be updated singularly.

This action allows you to update up to all media files immediately as a single command.

 

Instructions

Follow these steps to refresh your media files.

•        Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.

•        In Ambassador, select the Storyline Tools panel.

•        Click Open File.

•        Open the Storyline file containing the media names to be updated.

•        Click OK.

•        In the Active Command drop-down list, select Refresh Media From Sources.

•        Click Run Command.

•        Select the media assets to be refreshed. You can use the Select All and Select None buttons, use [Ctrl][Click] to toggle selections, or press [Shift][Click] to expand or decrease the selection.

•        Click OK.

•        Click Save File to commit the changes to the Storyline file.

•        Click Close File to release the file to Storyline.

 

Activity Description

Following is an example of the Refresh Media From Sources panel.

 

Key

Name

Description

A

Media List

List of items. All of the selected items will be updated. In the example above, the MP3 and MP4 files will be updated.

B

Select All

Select all items in the list.

C

Select None

Unselect all items.

D

Cancel

Cancel the operation and return the Storyline Tools panel without making any changes.

E

OK

Refresh the assets in the Storyline file from the selected source files.
