Storyline - Improve Video

Override the quality of the videos published by Storyline.

 

Background Information

Storyline's handling of a video's quality often results in inferior quality output. Although a number of explanations have been offered as to why it happens, and indeed, there have been found to be numerous unrelated causes for bad video output, you can use Ambassador's Improve Video action to use your own master copy of the source video to bypass the issue altogether.

 

The following article 'Video Quality bad when publishing in Storyline' was found in the public Storyline forum at: https://community.articulate.com/discussions/articulate-storyline/video-quality-bad-when-publishing-in-storyline.

 

Instructions

This action works on the published output version of your course. In general, you will run the Improve Video action just before uploading the course to the target location.

Follow these steps to improve the output video.

  • Publish your Storyline file to Review 360 using the option to Publish locally for manual upload; to Web; or to LMS.
  • Save and close your Storyline file in Storyline. Ambassador cannot access files when they are open in the Storyline application.
  • In Ambassador, select the Storyline Tools panel.
  • Click Open File.
  • Open the Storyline file containing the media names to be updated.
  • Click OK.
  • In the Active Command drop-down list, select Improve Video.
  • Click Run Command.
  • In the Video Folder textbox, enter the full path to the source videos on the project.
  • In the Output Folder textbox, enter the full path to the published output.
  • Click OK.
